## Rankmirrors

    #   sudo cp /etc/pacman.d/mirrorlist /etc/pacman.d/mirrorlist.backup

If you are running as root, run:

    # rankmirrors -n 6 /etc/pacman.d/mirrorlist.backup > /etc/pacman.d/mirrorlist

To use sudo, run this instead:

sudo rankmirrors -v -n 6 /etc/pacman.d/mirrorlst.backup | sudo tee /etc/pacman.d/mirrorlist

























To enable WiFi on VB:

First, get the name of your connection using:

    #   ip link

Should be something like `enp0s3`

Run
    #   nano /etc/wpa_supplicant.conf

And add:

    network={
        ssid="Haiku House"
        psk="password"
    }

Then run:

    #   ip link set dev enp0s3 up
    #   wpa_supplicant -B -i enp0s3 -c /etc/wpa_supplicant/wpa_supplicant.conf
    #   dhcpcd enp0s3


Verify that you connected successfully: 

    #   iw dev enp0s3 link
